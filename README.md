# Sandfold Studio - Brand Website Redevelopment

[Visit Live Site](https://sandfoldstudio.co.uk)

## Overview

**Sandfold Studio** is a web development and PPC marketing studio. I was tasked with redeveloping their brand website to provide a modern, interactive, and highly functional online presence. This project showcases my expertise in **Framer**, **React**, and **TypeScript** for building custom components and enhancing user experience.  

The goal was to merge design flexibility with dynamic functionality, creating a site that is visually appealing, fast, and scalable.

## My Role

- Redeveloped the Sandfold Studio website using **Framer** as the primary platform.
- Built **custom React and TypeScript components** to implement dynamic features beyond the platform’s default capabilities.
- Collaborated with the team to ensure the website reflected brand identity and marketing goals.
- Optimized site for **performance, accessibility, and responsiveness** across devices.
- Integrated interactive sections for case studies, services, and client engagement.

## Tech Stack

- **Platform:** Framer  
- **Languages:** TypeScript, JavaScript, HTML5, CSS3  
- **Frameworks/Libraries:** React, Framer Motion  
- **Tools:** Git, Figma (design collaboration), VS Code  

## Key Features

- Fully redeveloped **Framer-based website** with custom components.
- Dynamic **interactive sections** for portfolio, services, and client testimonials.
- Mobile-first **responsive design** with smooth animations.
- Optimized for **performance, SEO, and accessibility**.
- Seamless integration of Framer and custom React components for advanced functionality.

## Challenges & Solutions

- **Challenge:** Extending Framer’s default capabilities to support custom interactive components.  
  **Solution:** Developed reusable React + TypeScript components and integrated them within Framer to enhance functionality without compromising performance.  

- **Challenge:** Maintaining a visually rich, animated experience while keeping load times fast.  
  **Solution:** Applied performance optimization techniques, lazy loading, and lightweight animations using Framer Motion.

## Outcome

The redeveloped website successfully launched, providing a modern, interactive, and professional web presence for Sandfold Studio. The client highlighted the improved user engagement, performance, and flexibility of the new site compared to the previous version.
